# Why this project?

This example focuses on request validation in Spring Boot.

# Build and run test

## Build

```shell script
$ ./mvnw clean compile
```

## Run tests

```shell script
$ ./mvnw clean test
```

## Build package

```shell script
$ ./mvnw clean package
```
